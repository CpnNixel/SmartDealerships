# SmartDealerships
PoC application powered with .NET 6 using
 - PostgreSQL
 - EF Core
 - [FastEndpoints](https://github.com/dj-nitehawk/FastEndpoints)
 - [MediatR](https://github.com/jbogard/MediatR)

and with Frontend built on React using:
 - create-react-app
 - n18next
 - axios
 - formik
 - react-router
 - react-bootstrap
 - yup
